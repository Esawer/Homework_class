### 🚩 Project Overview
This project focuses on **Permissions** and **Cloud Storage**. It simulates a "Student vs. Teacher" relationship. 

**The Concept:** Teacher can create classes and homeworks to the particular class - student can join the a via class code (simmilar to MS Teams).
Students upload images of their homework, while Teachers log in to a separate dashboard to view those images and "grade" them (submitting data about the work).
